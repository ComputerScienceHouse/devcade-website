About the Gamejam

The Devcade team is hosting a week-long gamejam at RIT's MAGIC Center! The event is open to all RIT students, regardless of major, and will consist of in-person working sessions, workshops, and seminars.

You're encouraged to think outside of the box with your game! Some existing Devcade games include remakes of old arcade games, while some can be best classified as art projects. All are welcome to be submitted for the gamejam.

All participants are encouraged to build their games using either the Monogame framework (C#) or the Bevy engine (Rust). To learn more about using Devcade with Monogame or Bevy, check out the Getting Started page on the Devcade Docs.

There are $700 of prizes available to win, including an Elgato StreamDeck+, Raspberry Pi 4Bs, a Wacom Drawing Tablet, and many more!

How can I participate?

The gamejam has already started but it's not too late to start participating! Anyone can join the gamejam Discord server to ask questions and stay updated.

Join the Discord!
